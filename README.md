# 再切断仕損理由 記録アプリ

関西工場 フライスチーム向け、再切断仕損理由の記録・集計アプリです。

## 使い方

1. 理由コード（例: `1`、`8-①`、`15`）を入力
2. 発生日を確認・修正
3. 必要に応じて備考を入力（コード17・18は必須）
4. 「登録する」ボタンを押す
5. 月別の円グラフと一覧で確認できます

## GitHub Pages でiPhoneから使う手順

1. GitHubで新しいリポジトリを作成（例: `saikiDan-kiroku`）
2. このフォルダの `index.html` をアップロード（またはgit push）
3. リポジトリの **Settings → Pages → Source: Deploy from a branch → main → Save**
4. 数分後に `https://あなたのユーザー名.github.io/saikiDan-kiroku/` でアクセス可能
5. iPhoneのSafariでそのURLを開き、「ホーム画面に追加」でアプリのように使えます

## 注意事項

- データはブラウザの `localStorage` に保存されます
- **PCとiPhoneでデータは共有されません**（デバイスごとに独立）
- ブラウザのデータを削除すると記録が消えます → 定期的にCSVダウンロードを推奨
- 過去12ヶ月分の記録を遡って確認できます
